# MathSheety PRD - 一年级口算练习纸生成器

## 1. 产品概述
MathSheety 是一个纯前端的一年级口算练习纸生成工具。家长可自定义题型、范围与数量，一键生成适合 A4 打印的口算练习纸。

## 2. 核心功能
*   **参数配置**：
    *   **题型**：加法、减法、加减混合。
    *   **数字范围**：10以内、20以内、100以内（计算结果不得超过设定范围）。
    *   **题目数量**：20题、50题（默认）、100题。
    *   **进退位设置**：加法支持“允许/禁止进位”；减法支持“允许/禁止退位”。
*   **题目生成逻辑**：
    *   减法不出现负数。
    *   算式随机生成且尽量避免重复。
    *   格式：`8 + 7 = ____`
*   **答案控制**：一键切换“显示/隐藏答案”（默认隐藏）。
*   **A4 打印**：一键调用 `window.print()`。打印时仅保留练习纸区域（4或5列排版），自动隐藏配置面板与控制按钮。

## 3. 页面结构
*   **配置面板**（侧边栏或顶部）：包含标题、所有配置项、生成/重新生成按钮、显示/隐藏答案按钮、打印按钮。
*   **练习纸区域**（主视图）：
    *   页眉：标题（一年级口算练习纸）、姓名、日期、用时。
    *   题库网格：多列网格化布局（适合 A4 纸排版）。

## 4. 核心计算规则
*   **加法禁止进位**：个位数相加之和不超过 9（如 `12 + 3` 允许，`8 + 7` 禁止）。
*   **减法禁止退位**：被减数个位需大于或等于减数个位（如 `18 - 5` 允许，`15 - 8` 禁止）。

## 5. 技术方案
*   **技术栈**：纯 Vanilla HTML、CSS、JavaScript（单文件 `index.html`，无任何框架、后端或数据库依赖）。
