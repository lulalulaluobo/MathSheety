# MathSheety - 口算练习纸生成器设计文档

*   **创建日期**：2026-06-05
*   **状态**：已确认设计方案，准备进入开发实施阶段

---

## 1. 产品概述

MathSheety 是一个纯前端、免服务器的口算练习纸生成工具。家长可通过自定义题型、范围与数量，一键生成结构规整、适合 A4 打印的口算练习纸。

---

## 2. 界面与交互设计

### 2.1 整体布局 (Layout)
采用 **上下布局**：
*   **顶部配置面板**：包含项目标题、题型选择、范围选择、进退位复选框、题目数量选择，以及“生成题目”、“显示/隐藏答案”和“打印”三大动作按钮。
*   **底部居中练习纸**：在网页端以卡片阴影形式模拟标准 A4 纸张，保持水平居中，提供极佳的所见即所得 (WYSIWYG) 预览效果。

### 2.2 视觉风格 (Theme Style)
采用 **温馨护眼风 (Warm Pastel)**：
*   **背景色**：暖调象牙白/沙滩色（`#FAF6F0`），在屏幕端阅读时对家长的眼睛友好。
*   **点缀色**：莫兰迪色系，包括柔和的珊瑚橙（`#C88C6B` 作为主按钮色）、温润的森绿（`#8EA68B` 用于显示答案按钮）、科技蓝（`#6B9EC8` 用于打印按钮）。
*   **圆角与阴影**：面板采用大圆角（`12px`）和非常微弱的柔和阴影，增加界面高级感。

### 2.3 卷面排版 (Sheet Typography)
为保证打印出来纸面整洁，采用极致整齐排版方案：
*   **无序号设计**：彻底移除了题目的 `1) `、`2) ` 等前置序号，完全呈现为纯算式。
*   **4 列等宽网格**：题目列表分成 4 列排版，列与列之间使用轻量级的虚线分割线（`1px dashed #E6DFD5`）隔开，确保左右缝隙均等。
*   **单行与等号右对齐**：
    *   每道算式强制单行显示，禁止任何折行（`white-space: nowrap`）。
    *   在每一格内部使用 Flexbox 左右两端撑开（`justify-content: space-between`），使左侧的算式（如 `12 + 6`）靠左，等号与下划线占位符（`= ____`）在右侧完全垂直对齐。
*   **A4 垂直铺满**：使用 Flex 垂直对齐，使得题目列表能随着所选数量和纸张高度，在垂直方向上均匀地平铺满整张 A4 纸，避免底部留下大面积空白。

## 3. 技术实现方案

### 3.1 文件组织结构
项目采用纯 Vanilla 开发（无任何打包构建工具、无 React/Vue，以便家长可以直接双击离线运行），文件分为：
*   `index.html`：定义基本 DOM 骨架。
*   `style.css`：处理配置卡片、A4 纸张模拟效果、格网分割线样式，以及打印专用的 `@media print` 样式。
*   `app.js`：包含口算题目生成引擎、排版数据填充及事件监听器。

### 3.2 打印媒体查询 `@media print`
通过 CSS 确保打印时只渲染纯净的练习纸：
*   隐藏 `.config-panel` 和所有动作按钮。
*   将网页的象牙色背景重置为纯白，移除练习纸卡片阴影和外边框。
*   将练习纸容器的宽度设为 `100%`，利用浏览器内置的 A4 打印尺寸自适应平铺。

## 4. 口算题目生成算法

### 4.1 核心生成逻辑
*   **加法限制**：$a + b \le N$。若**禁止进位**，需满足 $(a \pmod{10}) + (b \pmod{10}) < 10$。
*   **减法限制**：$a - b \ge 0$，且被减数 $a \le N$。若**禁止退位**，需满足 $(a \pmod{10}) \ge (b \pmod{10})$。
*   **加减混合**：随机以 50%/50% 概率派发加法或减法。

### 4.2 去重机制
*   使用 JavaScript `Set` 记录生成的题目表达式字符串。
*   当生成新题时，比对 `Set`，若已存在则重新生成，直到满足所需数量，确保卷面上无完全相同的题目。
*   为了防止参数设置过窄（如：10以内不进位加法生成50题）导致死循环，算法在单次尝试超过 200 次时将跳出循环，允许极个别重复题。
